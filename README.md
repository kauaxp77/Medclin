# 🏥 MedClin - Sistema de Gestão Clínica

> **PCC (Projeto Complementar de Curso)** desenvolvido para as disciplinas de Linguagem de Programação II e Design Web.

O **MedClin** é uma solução robusta para a organização de fluxos de trabalho em clínicas e consultórios. O foco central é a otimização do agendamento de consultas e o gerenciamento centralizado de dados de pacientes e profissionais de saúde.

## 📖 Sobre o Projeto
O sistema foi projetado para atender três perfis distintos de usuários, garantindo uma interface específica para cada necessidade:

*   **📍 Pacientes:** Facilidade para agendar e acompanhar suas consultas.
*   **🩺 Equipe Médica:** Acesso rápido à agenda diária para organização de atendimentos.
*   **🏢 Recepção:** Gestão completa dos dados de médicos e pacientes, além do suporte para confirmação e desmarcação de horários.

---

## ⚙️ Funcionalidades
O MedClin implementa operações completas de **CRUD** (Create, Read, Update, Delete) para os seguintes módulos:

*   👤 **Perfis:** Gestão de níveis de acesso.
*   👥 **Usuários:** Cadastro de pacientes, médicos e recepcionistas.
*   📂 **Categorias:** Organização de especialidades e áreas médicas.
*   📝 **Artigos & Avaliações:** Sistema integrado para compartilhamento de conhecimento e feedback.

# 🏥 MedClin - Sistema de Gestão Clínica

> **PCC (Projeto Complementar de Curso)** desenvolvido para as disciplinas de Linguagem de Programação II e Design Web.

O **MedClin** é uma solução robusta para a organização de fluxos de trabalho em clínicas e consultórios. O foco central é a otimização do agendamento de consultas e o gerenciamento centralizado de dados de pacientes e profissionais de saúde.

## 📖 Sobre o Projeto
O sistema foi projetado para atender três perfis distintos de usuários, garantindo uma interface específica para cada necessidade:

*   **📍 Pacientes:** Facilidade para agendar e acompanhar suas consultas.
*   **🩺 Equipe Médica:** Acesso rápido à agenda diária para organização de atendimentos.
*   **🏢 Recepção:** Gestão completa dos dados de médicos e pacientes, além do suporte para confirmação e desmarcação de horários.

---

## ⚙️ Funcionalidades
O MedClin implementa operações completas de **CRUD** (Create, Read, Update, Delete) para os seguintes módulos:

*   👤 **Perfis:** Gestão de níveis de acesso.
*   👥 **Usuários:** Cadastro de pacientes, médicos e recepcionistas.
*   📂 **Categorias:** Organização de especialidades e áreas médicas.
*   📝 **Artigos & Avaliações:** Sistema integrado para compartilhamento de conhecimento e feedback.

## 💻 Como Executar o Projeto

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/seu-usuario/pcc_medclin_2024.git](https://github.com/seu-usuario/pcc_medclin_2024.git)
    ```
2.  Certifique-se de que o **XAMPP** esteja instalado e os módulos **Apache** e **MySQL** estejam ativos.
3.  Mova a pasta do projeto para o diretório `htdocs` do XAMPP.
4.  Importe o banco de dados via **PHPMyAdmin**.
5.  Acesse no navegador: `http://localhost/pcc_medclin_2024`
